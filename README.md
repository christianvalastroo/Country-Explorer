# Compito a sorpresa: Country Explorer

Gli studenti devono creare una piccola app React che permette di cercare paesi tramite API, visualizzare una lista di risultati e salvare alcuni paesi tra i preferiti usando React Context.

## API consigliata
`https://restcountries.com/v3.1/name/italy`
esempio
`https://restcountries.com/v3.1/name/france`


Obiettivo dell’app

L’app deve permettere di:

cercare un paese tramite input;
fare una chiamata API;
mostrare i risultati in una lista;
passare i dati ai componenti tramite props;
salvare/rimuovere paesi dai preferiti usando React Context;
mostrare una sezione “Preferiti”.
